# Study Timer CLI (Assignment 2)

## Project Description

Study Timer CLI is a simple command-line application designed to help students track their study sessions using a Pomodoro-style timer.

The application allows users to start study sessions and view statistics about the total number of sessions and study time recorded.

---

## Features

- Start a study timer
- Record study sessions automatically
- View total study sessions
- Calculate total study time

---

## Requirements

- Python 3.x installed
- Command line or terminal

---

## How to Run the Project

Open a terminal and navigate to the project folder:

cd study-timer-project

Start a study session:

python study_timer.py start

View study statistics:

python study_timer.py stats

---

## Example Output

Study session started...
Timer: 25 minutes
Session completed!
